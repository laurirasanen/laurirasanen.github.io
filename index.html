About me

I'm a programmer who makes video games and other assorted things. Currently studying game programming at Kajaani University of Applied Science.

I started programming as a hobby around 2012 and have been expanding my knowledge since. My strongest knowledge is in Unity and C#, but also often fiddle around with C++, Java, JavaScript and Python.

Besides programming, in my free time I play video games, watch way too many movies, listen to music usually around the clock, and exercise to keep the reaper at bay.

jtimer

Team Fortress 2 plugin and API for rocketjump speedrun servers written in Python.

The plugin uses Source.Python to interface with the engine.
The API uses Flask, SQLAlchemy along with various other modules and features protected endpoints with JWT authentication.
The API is designed to run on AWS Elastic Beanstalk and use RDS for the MySQL database.

https://github.com/occasionally-cool/jtimer https://github.com/occasionally-cool/jtimer-api
