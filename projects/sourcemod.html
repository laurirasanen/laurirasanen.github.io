Sourcemod plugins

A collection of sourcemod plugins I've made or helped make for Team Fortress 2.
Centered around the jumping community and Tempus Network.

sm-translate

A TF2 plugin for real-time chat translation using Google Cloud Translation API.
Source: https://github.com/laurirasanen/sm-translate/

Rampbugfix

A plugin that I've rewritten countless times trying to make it work.
The idea was to prevent ramp bugs, which occasionally make you stop suddenly when sliding along a ramp, caused by some issue with the Source engine.
The plugin itself technically works, but something about how Source engine works will occasionally give you a massive speed boost if you happen to rampbug and shoot a rocket at the same time.
Source: https://github.com/laurirasanen/rampbugfix/

Stopstickybounce

A plugin intended for stopping stickybombs from bouncing off other players.
Even though stickybombs should not bounce off your teammates, occasionally they still do. This usually happens if you hit them in the head.
This little plugin makes stickies pass straight through teammates and even enemies if you so wish.
Intended for the jumping community, because it's annoying to have your sticky fly to God knows where when there are multiple people on the same part of the map.

Made in collaboration with Jayess and JoinedSenses.
https://github.com/jsza/
https://github.com/joinedsenses

Source: https://github.com/laurirasanen/stopstickybounce/

Detectfreezeglitch

A plugin for detecting freezeglitches.
Freezeglitching is when a player purposefully lags themselves mid-air to gain an unfair speed boost.
Intended for the Tempus jump network, because freezeglitching to gain an advantage while speedrunning is pretty low.
The idea was to automate detection of freezeglitched runs, so that people wouldn't get away with doing it.
Not intended to be an automated ban system but rather a run flagging system.
Will still require an admin to watch the STV demo after the fact, but should decentivise people from doing it and stop them from getting away with it.
Tempus developer, Jayess, wanted to turn this into a prevention system, where players would not be able to shoot for a while after freezing.
However, this solution was screwing over people with laggy computers/connection and was rolled back.
Currently the plugin is ready to go, but lacking event hooking/implementation on Tempus backend.

Made in collaboration with Jayess.
https://github.com/jsza/

Source: https://github.com/laurirasanen/detectfreezeglitch/
